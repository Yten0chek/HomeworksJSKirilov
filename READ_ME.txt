READ_ME for RLE

Программа в своей дерриктории ищет файл с именем input.txt, который она обробатывает,
сжимая при помощи RLE сжатия, все сжатые и несжатые(по ситуации) ряды повторяющихся символов сразу же записываются в файл output.txt

Далее, программа считывает в своей дерриктории ранее созданный файл output.txt,
из которого декодирует RLE сжатие, записывая декодированные символы в файл decode.txt

В трёх тестах представлено 3 разных случая, 1 - когда шифруется спец символ #
2 - обычная строка с несколькими повторениями
3 - 426 букв a, для показа, если количество символов > 255
